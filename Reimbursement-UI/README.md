# Reimbrsement UI

This is the frontend applicaiton. It uses React and typescript.

## Local Setup

1. Clone the repo.
```bash
$ git clone https://github.com/danieljhkim/Reimbursement-App.git
```

2. Install dependencies.
```bash
$ cd Rimbursement-UI
$ npm install
```

3. Start the application.
```bash
$ npm run start
```
